Generate comprehensive, well-structured documentation for the specified code, feature, or system.

Use the doc-agent to:
- Write clear, concise technical documentation
- Follow Markdown best practices
- Include code examples where helpful
- Ensure proper structure with headings, lists, and formatting
- Add necessary context and explanations

The agent will analyze the codebase, understand conventions, and produce documentation that matches the project's style.
