Conduct a thorough code review focusing on quality, security, performance, and best practices.

Use the review-agent to:
- Identify bugs, security vulnerabilities, and code smells
- Check for performance issues and optimization opportunities
- Verify adherence to coding standards and best practices
- Suggest improvements with specific code patches
- Provide actionable feedback with severity ratings

The agent will analyze changes, run static analysis, and produce a detailed review report with suggested fixes.
