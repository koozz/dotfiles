Analyze application performance and identify optimization opportunities.

Use the performance-engineer to:
- Profile application performance and identify bottlenecks
- Analyze memory usage, CPU utilization, and I/O patterns
- Implement caching strategies and optimize database queries
- Set up observability with OpenTelemetry and monitoring
- Optimize Core Web Vitals and improve user experience

The agent will investigate performance issues, provide detailed analysis, and deliver actionable optimization recommendations.
